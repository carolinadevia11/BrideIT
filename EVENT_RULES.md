# Rules for Calendar Events

This document outlines the rules and conditions for swapping and moving calendar events.

## Rules for Swapping Events

For two events to be swapped, the following conditions must be met:

1.  A swap request must be initiated for one of the events.
2.  The event being swapped with must be explicitly selected.
3.  The parent who did *not* create the swap request must approve it. A parent cannot approve their own request.
4.  Both events must exist in the family's calendar.

## Rules for Moving Events

For an event to be moved (modified) to a new date, the following conditions must be met:

1.  A move (modify) request must be created for the event.
2.  A new date for the event must be provided with the request.
3.  The parent who did *not* create the move request must approve it. A parent cannot approve their own request.
4.  The event must exist in the family's calendar.
